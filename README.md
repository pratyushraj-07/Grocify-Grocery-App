# Grocify-Grocery-App

Grocify is a grocery shopping Android application built using Kotlin and XML in Android Studio.

## Features

- OTP Login Screen
- Product Listing using RecyclerView
- Search and Filter Products
- Add to Cart Functionality
- Checkout / Order Summary Screen
- Multi-screen Navigation

## Tech Stack

- Kotlin
- XML
- RecyclerView
- Android Studio
- Material Design Components

## How to Run

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle files
4. Run the app on an emulator or physical Android device

## Screens

- Login Screen
- Home Screen
- Cart Screen
- Checkout Screen
